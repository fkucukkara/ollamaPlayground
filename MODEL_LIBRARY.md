## Model Categories

1. **General Purpose**
   - For chat and text generation
   - Best for everyday use

2. **Code Generation**
   - Specialized in programming
   - Support multiple languages

3. **Vision Models**
   - Can analyze images
   - Respond to visual queries

4. **Reasoning Models**
   - Enhanced logical capabilities
   - Better at complex problem solving
